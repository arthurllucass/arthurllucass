### 📌 Projetos Destacados

#### 🚚 Transportadora GynLog  
🔗 [Repositório no GitHub](https://github.com/arthurllucass/Transportadora-GynLog.git)

Aplicação **desktop em Java** para **gestão de frotas**, desenvolvida com foco em regras de negócio e organização de código.  
O sistema permite o controle de veículos, registro de despesas, geração de relatórios financeiros e exportação de dados em **PDF**.

**Principais funcionalidades:**
- Cadastro e gerenciamento de veículos  
- Registro e controle de despesas operacionais  
- Geração de relatórios financeiros  
- Exportação de relatórios em PDF  

**Tecnologias utilizadas:**
- Java 21  
- Java Swing  
- Arquitetura MVC  
- Manipulação de arquivos (`.txt`)  
- Geração de PDF (OpenPDF)  
- Datas e horários (`Date`, `LocalDate`, `SimpleDateFormat`)  
- Programação Orientada a Objetos (POO)
